# Cita Previa Padrón Leganés

Sistema de monitorización de citas disponibles para padrón en el Ayuntamiento de Leganés.

## Descripción
Web tipo landing donde los usuarios registran sus datos de contacto para recibir notificaciones cuando haya citas disponibles en https://intraweb.leganes.org/CitaPrevia/

## Características
- Verificación automática cada 30min (8-15h) y cada 1h (resto)
- Notificaciones por Email, Telegram y WhatsApp
- Sin login - registro simple de datos de contacto

## Tecnologías
- Backend: Python (FastAPI)
- Frontend: HTML/CSS/JS
- Scraper: Selenium
- Base de datos: SQLite
